# REST API with Node.js and Express

A simple REST API built with Node.js and Express.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. For development with auto-reload:
```bash
npm run dev
```

## API Endpoints

- `GET /` - Welcome message

## Technologies Used
- Node.js
- Express
- CORS
- dotenv
